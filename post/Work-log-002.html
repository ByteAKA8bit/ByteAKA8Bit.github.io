工作小记——入职第二天
2022-03-31

迟到

受第一天的负面情绪影响，今天早上惊醒已经是 8 点 25 分，虽说公司离得不是很远，但是也是要迟到了。

磨蹭的洗了头，牙都没刷就去赶地铁。

急急忙忙出错了地铁口，迟到了 5 分钟。

上午

上午客户反馈修改意见了，很烦这种拉三四个微信群说事的情况。

写代码的就两个人，拉了三个群说事，我们两个人包围了十几个人。

就是又给出了四条修改意见。

有一个高德地图锁定搜索范围的小问题，我信誓旦旦的告诉别人这个没办法…

下午

下午有人找我解决地图的事。

发现了高德有锁定搜索范围的属性…

脸都红了

记一记

var placeSearch = new AMap.PlaceSearch({
  type: '餐饮服务', // 兴趣点类别
  pageSize: 5, // 单页显示结果条数

});

我慌了啊，下午这事干的。

后端把一个导入功能前后端都做完了，我还在说地图不能改…

晚上跑到凯德 MALL 想找点吃的，结果没找到
